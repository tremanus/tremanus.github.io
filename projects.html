ScholaLatinae
I created a Latin website, a 501(c)(3) registered nonprofit, that teaches people the Classics for free through resources ranging from textbooks to study guides, lessons found on YouTube, practice questions with a leaderboard, and games such as Geoguessr map of the Classical World. I built the website through Squarespace, and I used HTML, CSS, and JavaScript to build plugins and customize the website.
Practice Questions - ScholaLatinae
Using Flask (Python) for the back-end and HTML, CSS, and JavaScript for the front-end, I fully developed a practice question system used for ScholaLatinae & any form of practice quizzes. The code takes 10 random questions from a JSON database and presents them in a user friendly manner for people to answer, and their scores go onto a leaderboard that is automatically updated. I used Vercel to host the website and to create a Postgres database for the leaderboard system.
